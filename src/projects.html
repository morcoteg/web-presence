CourseMatcher - Web Application

Created a Java EE web application using the Apache Struts framework in a team environment. A partner and I developed a simple web application that would provide students the ability to search for a course they were taking, and obtain a list of all the students taking the same course. The student could then add themselves to a courses or remove themselves. If logged in as an administrator, you have the ability to add new professors, delete a retiring/leaving (or fired) professor, and assign/unassign professors to specific courses. The application would originally use servlets to access a database, query that database for the students taking the same course based on what the user input was, and return the results with the students' emails displayed. Now CourseMatcher runs on the Struts framework, and we achieve the same end result by using Action classes and Action Forms, for example for easier form validation (rather than using javascript). HTML/CSS were used for styling and layout purposes. jQuery was used for adding animation/responsive features to our application.
